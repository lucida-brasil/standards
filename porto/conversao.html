Identificando conversões de sucesso

Para identificarmos no Google Analytics as conversão de sucesso da Hypnobox precisamos ter acesso ao id do cliente ou id da conversão gerado pela ferramento, essa informação pode ser passada para o GA pelo dataLayer do Google Tag Manager utilizando um evento.

dataLayer.push({ 'event' : 'hypnobox.sucesso', 'id' : '00000000' });

Esse pequeno bloco de código sinaliza o GTM que houve uma conversão, quando isso ocorre nossos scripts podem ler o atributo id que deve conter o id gerado pela ferramenta e efetuar o disparo para o GA, a vantagem desse método é que futuras tags também podem ser associadas a conversão.

O mesmo método pode ser utilizado para outras conversões que gerem id's no back-end, sempre que o usuário receber o feed-back de sucesso disparamos um evento para o GTM informando a ferramenta ou processo que gerou a conversão e qual o id da conversão.
